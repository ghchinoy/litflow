# Examples

Explore the capabilities of LitFlow through these live interactive examples.

## Basic Features

- **[Overview](overview/)**: A comprehensive demo of all core features.
- **[Basic Flow](basic/)**: The simplest setup for a dynamic flow.
- **[Static Display](static-display/)**: Using LitFlow as a read-only diagram for documentation.

## Customization

- **[Custom Nodes](custom-nodes/)**: Building rich, interactive nodes with custom logic.
- **[Custom Edges](custom-edges/)**: Creating bespoke edge styles and paths.
- **[Edge Markers](edge-markers/)**: Showcasing arrowheads and bidirectional edges.
- **[Marquee Selection](marquee-selection/)**: Selecting multiple nodes and edges by dragging a box.
- **[Multiple Handles](multiple-handles/)**: Managing complex connectivity with multiple target and source handles.

## Advanced Patterns

- **[Subflows & Groups](subflows/)**: Nesting nodes and managing hierarchical structures.
- **[Dynamic Interactivity](dynamic-interactivity/)**: Responding to events and updating the graph in real-time.

## Tools & Editors

- **[Inspector Designer](designer/)**: A full-featured authoring tool with a drag-and-drop palette, property inspector, and live JSON sync.
- **[Schema-Driven Node](schema-node/)**: A powerful node type that auto-generates its UI from a JSON Schema, simulating a Generative AI execution flow.

---

### Run locally
To explore these examples and their source code locally:
```bash
git clone https://github.com/ghchinoy/litflow.git
cd litflow
pnpm install
pnpm dev
```
